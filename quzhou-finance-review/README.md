# 衢州发展财经复盘 HyperFrames 项目

This project is a 90-second, 12-scene HyperFrames composition for a dark financial documentary recap. It uses only synthetic infographic animation and fixed storyboard price data.

## Files

- `index.html` - main 1920x1080 HyperFrames composition, duration 90 seconds.
- `DESIGN.md` - locked visual identity and constraints.
- `narration.zh-CN.txt` - Chinese voiceover script.
- `subtitles.srt` - Chinese subtitles.
- `docs/timeline.md` - timeline preview.
- `docs/audio-plan.md` - male voiceover, music, and mix guidance.
- `docs/final-render-checklist.md` - checklist before final MP4 render.
- `docs/preview-status.md` - current local preview status and environment notes.
- `docs/draft-render-guide.md` - audio placement and draft render commands.
- `preview-static.html` - static local preview page that does not require HyperFrames CLI.
- `assets/scene-preview.svg` - scene preview board.
- `assets/timeline-preview.svg` - visual timeline preview.
- `assets/audio/voiceover.zh-CN.male.wav` - final male voiceover target path, currently missing.
- `assets/audio/bgm.cinematic.mp3` - final background music target path, currently missing.

## Locked Prices

- 2019-2021: `2.80元 → 5.13元`
- 2021-2024: `5.13元 → 1.58元`
- 历史真实成交最低价: `1.58元`
- 盘中前复权低点: `1.55元（2024/7/8）`
- 2024-2025: `1.58元 → 6.03元 → 3.50元`
- 2026至今: `5.51元 → 2.76元`

## Scene Count

The preview is aligned to the original 12-panel storyboard from `00:00-00:05` through `01:28-01:30`. It intentionally includes the separate lowest-price scene, state-owned-capital scene, semiconductor spike scene, risk page, interaction page, and ending page.
